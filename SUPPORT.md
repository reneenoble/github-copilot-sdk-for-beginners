# Support

## How to Get Help

This repository contains a self-paced course. Here's how to get help:

### Course Content Issues

- **Something doesn't work?** Open a [Bug Report](https://github.com/microsoft/github-copilot-sdk-for-beginners/issues/new?template=bug_report.md).
- **Have a suggestion?** Open a [Content Suggestion](https://github.com/microsoft/github-copilot-sdk-for-beginners/issues/new?template=content_suggestion.md).

### GitHub Copilot SDK Questions

- [GitHub Copilot SDK Documentation](https://github.com/github/copilot-sdk)
- [GitHub Copilot SDK Discussions](https://github.com/github/copilot-sdk/discussions)
- [GitHub Copilot Documentation](https://docs.github.com/en/copilot)

### Community

- [Azure AI Community Discord](https://discord.gg/ByRwuEEgH4)

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
